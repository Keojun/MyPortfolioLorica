# Ken Eubert Lorica — IT Portfolio

A React portfolio site for internship and IT job applications. Built with Vite and React.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for deployment

```bash
npm run build
npm run preview
```

Deploy the `dist` folder to Vercel, Netlify, or GitHub Pages.

## Customize

- **Content:** Edit `src/data/portfolio.js`
- **Photo:** Replace `public/profile.png`
- **Styles:** Edit `src/index.css`
